1 面试题描述

以下代码运行后所得结果

/**
 * 局部变量与成员变量的区别
 */
public class Variables {
    static int s;        // 静态变量
    int i;                // 实例变量
    int j;                // 实例变量
    {
        int i = 1;        // 局部变量
        i++;            // 就近原则，局部变量i
        j++;
        s++;
    }
    public void test(int j){    // int j j 也是个局部变量
        i++;
        j++;        // 就近原则
        s++;
    }

    public static void main(String[] args) {
        Variables variables1 = new Variables();
        Variables variables2 = new Variables();
        variables1.test(10);
        variables1.test(20);
        variables2.test(30);
        System.out.println(variables1.i + "," + variables1.j + "," + variables1.s);
        System.out.println(variables2.i + "," + variables2.j + "," + variables2.s);
    }
}

2 试题考点

  • 就近原则
  • 变量的分类
    • 局部变量
    • 成员变量：类变量、实例变量
  • 非静态代码块的执行：每次创建实例对象都会执行
  • 静态代码块是在虚拟机加载类的时候，就执行的，而且只执行一次
  • 方法的调用规则：调用一次执行一次

3 static 关键字

  1. static关键字的用途

static方法就是没有this的方法。在static方法内部不能调用非静态方法，反过来是可以的。而且可以在没有创建任何对象的前提下，仅仅通过类本身来调用static方法。这实际上正是static方法的主要用途。

一句话描述就是：方便在没有创建对象的情况下来进行调用（方法/变量）

很显然，被static关键字修饰的方法或者变量不需要依赖于对象来进行访问，只要类被加载了，就可以通过类名去进行访问。因此在静态方法中不能访问类的非静态成员变量和非静态成员方法，因为非静态成员方法/变量都是必须依赖具体的对象才能够被调用。

static可以用来修饰类的成员方法、类的成员变量，另外可以编写static代码块来优化程序性能。

4 变量的分类

局部变量

方法体、代码块{}中的变量，使用前必须显式赋值

成员变量

  • 类变量

    也叫静态变量，是由static修饰的变量。静态变量被所有的对象所共享，在内存中只有一个副本，它当且仅当在类初次加载时会被初始化。

  • 实例变量

    未有static修饰的变量。实例变量是对象所私有的，在创建对象的时候被初始化，存在多个副本，各个对象拥有的副本互不影响。

局部变量和成员变量的区别

① 声明的位置

  • 局部变量：方法体{}中，形参，代码块{}中
  • 成员变量：类中方法外
    • 类变量：静态变量，被static修饰的
    • 实例变量：没有被static修饰的

② 修饰符

  • 局部变量：只能被final修饰
  • 类变量：可以被public、protected、private、final、static、volatile、transient修饰

③ 值存储的位置

  • 局部变量：栈
  • 成员变量
    • 类变量：方法区
    • 实例变量：堆

④ 作用域

  • 局部变量：从声明开始，到所属的 } 结束
  • 成员变量
    • 类变量：在当前类中“ 类名. ”(有时类名.可以省略)，在其他类中 “类名.” 或 “对象名.” 访问
    • 实例变量：在当前类中 “this.” (有时this.可以缺省)，在其他类中 “对象名.” 访问

⑤ 声明周期

  • 局部变量：每一个线程，每一次调用执行都是新的生命周期
  • 成员变量
    • 类变量：随着类的初始化而初始化，随着类的卸载而消亡，该类的所有对象的类变量是共享的
    • 实例变量：随着对象的创建而初始化，随着对象的被回收而消亡，每一个对象的实例变量是独私有的

局部变量与成员变量重名时，如何区分？

① 局部变量与实例变量重名

  • 在实例变量前面加 “this.”

② 局部变量与类变量重名

  • 在类变量前面加 “类名.”

5 试题解析

tips：

  • 静态变量是属于类的，是要存放在方法区中。

  • 实例变量是属于对象的，是要存放在堆中。

  • 局部变量是属于方法的，是要存放在局部变量表中。

  • 一个方法的参数、方法内定义的变量是存放在虚拟机栈中的局部变量表。

  • 方法内的参数都遵循就近原则，后序操作都是针对最近的那个参数进行的。

  • 一个方法的参数、方法内定义的变量，生命周期仅局限于本方法内，若无return传递出去，则仅在局部变量表中自生自灭。
