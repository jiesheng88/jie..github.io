【数据结构-二叉树】遍历的应用
数据结构 二叉树
二叉树 遍历
发布日期:   2021-04-14
阅读次数:  

一、考点

  • 考察应聘者对二叉树中序遍历的理解程度。
  • 考察应聘者分析复杂问题的能力。把构建二叉树的问题大问题分解成构建左、右子树的两个小问题，然后递归解决。
  • 考察应聘者分析复杂问题的能力。应聘者只有画出二叉树的结构图、通过具体的例子找出中序遍历下一个节点的规律，才有可能设计出可行的算法。(感觉上找规律很重要)

二、前、中、后序遍历及层次遍历

2.1 前序遍历

根——左——右

  • 递归写法
//前序遍历（递归）方法
public static void preOrderWithRecursion(HeroNode root) {
    HeroNode p = root;    // 定义一个移动指针

    // 递归结束条件
    if (p == null) {
        System.out.println("二叉树为空，无法遍历~~~");
    }

    // 递归体
    System.out.println(p);     // 根节点
    if (p.left != null) {    // 左子树
        preOrderWithRecursion(root.left);
    }
    if (p.right != null) {    // 右子树
        preOrderWithRecursion(root.right);
    }
}
  • 非递归写法

  利用栈，一直向左遍历，非空即入栈，同时加入列表；弹出栈顶，同样方法，遍历右子树

//前序遍历（非递归）方法
public static void preOrderWithStack(HeroNode root) {

    Stack<HeroNode> stack = new Stack<>();
    List<HeroNode> list = new ArrayList<HeroNode>();
    HeroNode p = root;    // 定义一个移动指针

    while (p!=null || !stack.isEmpty()) {    // 外循环条件
        // 一直向左遍历，非空即入栈，同时加入列表
        while (p != null) {        // 内循环条件
            list.add(p);
            stack.push(p);
            p = p.left;
        }

        // 弹出栈顶，遍历右子树
        HeroNode temp = stack.pop();
        p = temp.right;
    }

    // 输出前序遍历结果
    for (HeroNode heroNode : list) {
        System.out.println(heroNode);
    }

}

2.2 中序遍历

左——根——右

  • 非递归

  利用栈，一直向左遍历，非空即入栈；弹出栈顶，加入列表，同样方法，遍历右子树

//中序遍历（非递归）方法
public static void infixOrderWithStack(HeroNode root) {

    Stack<HeroNode> stack = new Stack<>();
    List<HeroNode> list = new ArrayList<HeroNode>();
    HeroNode p = root;    // 定义一个移动指针

    while (p!=null || !stack.isEmpty()) {
        // 一直向左遍历，非空即入栈
        while (p != null) {
            stack.push(p);
            p = p.left;
        }

        // 弹出栈顶，加入列表，遍历右子树
        HeroNode temp = stack.pop();
        list.add(temp);
        p = temp.right;
    }

    // 输出中序遍历结果
    for (HeroNode heroNode : list) {
        System.out.println(heroNode);
    }

}

注意：

  前序遍历和中序遍历代码几乎一样，不同仅仅是元素添加到列表的时机。

2.3 后序遍历

左——右——根

  • 非递归写法

  借助前序遍历，前序遍历是 根 左 右，先将前序遍历代码改成 根 右 左，然后反转即是后序遍历 左 右 根

  利用栈，一直向右遍历，非空即入栈，同时加入列表；弹出栈顶，同样方法，遍历左子树，最后列表倒叙输出。

// 后序遍历（非递归）方法
// 前序遍历是 根 左 右，先将前序遍历代码改成 根 右 左，然后反转即是后序遍历 左 右 根 
public static void postOrderWithStack(HeroNode root) {

    Stack<HeroNode> stack = new Stack<>();
    List<HeroNode> list = new ArrayList<HeroNode>();
    HeroNode p = root;    // 定义一个移动指针

    while (p!=null || !stack.isEmpty()) {
        // 一直向右遍历，非空即入栈，同时加入列表
        while (p != null) {
            list.add(p);
            stack.push(p);
            p = p.right;
        }

        // 弹出栈顶，遍历左子树
        HeroNode temp = stack.pop();
        p = temp.left;
    }

    // 输出后序遍历结果：倒叙输出列表即可
    for (int i = list.size()-1; i >= 0; i--) {
        System.out.println(list.get(i));
    }
}

2.4 层次遍历

借助队列实现

  • 先将二叉树的头结点入队列，然后出队列。出队列时，访问该节点。
  • 如果有左子树，则将左子树的根结点入队，然后出队列。出队列时，访问该节点。
  • 如果有右子树，则将右子树的根结点入队。然后出队列。出队列时，访问该节点。
  • 如此反复，直到队列为空为止。
// 完全二叉树层次遍历，结果即是顺序存储结果
/**
     * 借助队列实现 
     *     1.先将二叉树的头结点入队列，然后出队列。出队列时，访问该节点。
     *     2.如果有左子树，则将左子树的根结点入队，然后出队列。出队列时，访问该节点。
     *     3.如果有右子树，则将右子树的根结点入队。然后出队列。出队列时，访问该节点。 
     *     4.如此反复，直到队列为空为止。
     * @param root
     * @return
     */
public static List<HeroNode> levelOrder(HeroNode root) {
    HeroNode p = root;
    List<HeroNode> list = new ArrayList<HeroNode>();
    Deque<HeroNode> queue = new ArrayDeque<>();

    if (p == null) {
        System.out.println("空树~~~");
    }

    // 1.先将二叉树的头结点入队列
    queue.add(p);
    while (!queue.isEmpty()) { // 不能使用queue！=null来判断空值
        // 2.出队列，加入列表
        HeroNode tmp = queue.poll();
        list.add(tmp);

        // 3.判断左子树
        if (tmp.left != null) {
            queue.add(tmp.left);
        }
        // 4.判断右子树
        if (tmp.right != null) {
            queue.add(tmp.right);
        }
    }

    return list;
}

三、重建二叉树

3.1 题目描述

输入某二叉树的前序遍历和中序遍历的结果，请重建该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。

举例：

前序遍历 preorder = [3,9,20,15,7]
中序遍历 inorder = [9,3,15,20,7]

返回如下的二叉树：

    3
   / \
  9  20
    /  \
   15   7

限制：

0 <= 节点个数 <= 5000

3.2 题目解析

对于任意一颗树而言，前序遍历的形式总是

[ 根节点, [左子树的前序遍历结果], [右子树的前序遍历结果] ]

即根节点总是前序遍历中的第一个节点。

中序遍历的形式总是

[ [左子树的中序遍历结果], 根节点, [右子树的中序遍历结果] ]
  • 先根据前序序列的第一个数字创建根节点
  • 接下来在中序序列中找到根节点的位置
  • 确定左、右子树节点的数量
  • 在前序遍历和中序遍历序列中划分了左、右子树节点的之后，便可以递归的调用去分别构建它的左、右子树

优化：

可以考虑使用哈希表来帮助我们快速地定位根节点。对于哈希映射中的每个键值对，键表示一个元素（节点的值），值表示其在中序遍历中的出现位置。在构造二叉树的过程之前，我们可以对中序遍历的列表进行一遍扫描，就可以构造出这个哈希映射。在此后构造二叉树的过程中，我们就只需要 O(1)O(1) 的时间对根节点进行定位了。

3.3 代码实现

/**
* 二叉树节点
* public class TreeNode {
*     int val;
*     TreeNode left;
*     TreeNode right;
*     TreeNode(int x) { val = x; }
* }
*/
class Solution {
    private Map<Integer,Integer> indexMap;

    public TreeNode buildTree(int[] preorder, int[] inorder) {
        int n = preorder.length;
        // 构造一个哈希映射，帮助快速定位根节点在中序序列中的位置
        indexMap = new HashMap<Integer,Integer>();
        for(int i = 0; i < n; i++){
            indexMap.put(inorder[i], i);    // key：中序遍历的结果，value：结果对应的下标
        }
        return myBuildTree(preorder, inorder, 0, n-1, 0, n-1);
    }

/**
    * @param preorder            前序序列
    * @param inorder            中序序列
    * @param preorder_left        前序序列中开始的位置
    * @param preorder_right    前序序列中结束的位置
    * @param inorder_left        中序序列中开始的位置
    * @param inorder_right        中序序列中结束的位置
    * @return
    */
    public TreeNode myBuildTree(int[] preorder, int[] inorder, int preorder_left, int preorder_right, int inorder_left, int inorder_right) {
    // 递归结束条件
    if(preorder_left > preorder_right){
        return null;
    }

    // 前序遍历中的第一个节点就是根节点，获取其索引
    int preorder_root = preorder_left;
    // 在中序遍历中定位根节点，保存的都是下标值
    int inorder_root = indexMap.get(preorder[preorder_root]);
    // 获取前序序列中左子树中节点的数目
    int size_left_subtree = inorder_root - inorder_left;

    // 创建根节点
    TreeNode root = new TreeNode(preorder[preorder_root]);
    // 递归地构造左子树，并连接到根节点
    // 先序遍历中「从 左边界+1 开始的 size_left_subtree」个元素就对应了中序遍历中「从 左边界 开始到 根节点定位-1」的元素
    root.left = myBuildTree(preorder, inorder, preorder_left+1, preorder_left+size_left_subtree, inorder_left, inorder_root-1);
    // 递归地构造右子树，并连接到根节点
    // 先序遍历中「从 左边界+1+左子树节点数目 开始到 右边界」的元素就对应了中序遍历中「从 根节点定位+1 到 右边界」的元素
    root.right = myBuildTree(preorder, inorder, preorder_left+size_left_subtree+1, preorder_right , inorder_root+1, inorder_right);
    return root;
    }
}

3.4 利用迭代方法实现

/**
* 二叉树节点
* public class TreeNode {
*     int val;
*     TreeNode left;
*     TreeNode right;
*     TreeNode(int x) { val = x; }
* }
*/
class Solution {
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        if (preorder == null || preorder.length == 0) {
            return null;
        }
        TreeNode root = new TreeNode(preorder[0]);
        Deque<TreeNode> stack = new LinkedList<TreeNode>();
        stack.push(root);
        int inorderIndex = 0;
        for (int i = 1; i < preorder.length; i++) {
            int preorderVal = preorder[i];
            TreeNode node = stack.peek();
            if (node.val != inorder[inorderIndex]) {
                node.left = new TreeNode(preorderVal);
                stack.push(node.left);
            } else {
                while (!stack.isEmpty() && stack.peek().val == inorder[inorderIndex]) {
                    node = stack.pop();
                    inorderIndex++;
                }
                node.right = new TreeNode(preorderVal);
                stack.push(node.right);
            }
        }
        return root;
    }
}

四、二叉树的下一个节点

4.1 题目描述

给定一颗二叉树和其中的一个节点，如何找出中序遍历序列的下一个节点？

注意，树中的节点不仅包含左右子节点，同时包含指向父节点的指针。

/**
 * 二叉树节点
 */
public class TreeLinkNode {
    int val;
    TreeLinkNode left = null;
    TreeLinkNode right = null;
    // 父节点
    TreeLinkNode parent = null;

    TreeLinkNode(int val) {
        this.val = val;
    }
}

4.2 题目解析

画出二叉树的结构图、通过具体的例子找出中序遍历下一个节点的规律，才有可能设计出可行的算法。

通过对下一个节点的情况分析，主要有以下两种情况：

  • 如果一个节点的右子树不为空，那么该节点的下一个节点是右子树的最左节点。也就是说，从右子节点出发一直沿着指向左子节点的指针就能找到它的下一个节点。 如下图：

比如，当前节点是b，先到其右子节点e，再到左子节点g

  • 如果一个节点的右子树为空
    • 当前节点是其父节点的左子节点，则其下一个节点就是其父节点
    • 当前节点是其父节点的右子节点，那么需要沿着父节点的指针一直向上遍历，直到找到一个节点（该节点是其父节点的左子节点），那么该节点的父节点便是要找的下一个节点。

比如，当前节点是h，沿着父节点的指针向上遍历，先到达节点e，由于节点e是它父节点b的右子节点，继续向上遍历达到节点b。由于节点b是它父节点a的左子节点，因此节点b的父节点a就是节点h的下一个节点。

4.3 代码实现

/**
 * 二叉树节点
 */
class TreeLinkNode {
    String val;
    TreeLinkNode left = null;
    TreeLinkNode right = null;
    // 父节点
    TreeLinkNode parent = null;

    TreeLinkNode(String val) {
        this.val = val;
    }
}

/**
     * 寻找二叉树当前节点在中序序列中的下一个节点
     * @param pNode
     * @return
     */
public static TreeLinkNode getNext(TreeLinkNode pNode) {
    if (pNode == null) {
        return null;
    }

    // 中间指针
    TreeLinkNode pNext = null;
    // 1.当前节点有右子树的情况
    if (pNode.right != null) {
        // 从右子节点出发，一路向左
        TreeLinkNode pRight = pNode.right;
        while (pRight.left != null) {
            pRight = pRight.left;
        }
        pNext = pRight;
    }else if(pNode.parent != null){
        // 2.当前节点没有右子树情况
        TreeLinkNode pParent = pNode.parent;    // 当前节点的父节点
        TreeLinkNode pCurrent = pNode;            // 当前节点赋给一个中间变量
        // 2.1 当前节点是其父节点的左子节点
        if (pParent.left == pNode) {
            pNext = pParent;
        }
        // 2.2 当前节点是其父节点的右子节点
        while (pParent!=null && pParent.right == pCurrent) {
            pCurrent = pParent;
            pParent = pCurrent.parent;
        }
        pNext = pParent;
    }
    return pNext;

}
